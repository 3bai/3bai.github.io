js实现定时任务

发表于 2013-11-19   |   分类于 设计   |     |  

在日常的开发中，如果我们想要在某时执行某件操作的时候，大多数人的做法就是时钟时间，然后不断的判断是否到点等，这样的做法可谓中规中矩，但我认为却不是最佳的方案。

网上检索此类解决方案，代码均是一大堆，看的让人实在头疼，无奈只能自己动手丰衣足食了。我这个算法不一定是最好的，也不一定毫无bug，但我个人使用起来非常的顺手，如果有人发现了问题，那么自然是极好的。

代码无版权，随意修改和使用，毕竟我发这博文就是为了供大家参考和研究。

但我希望能在心底里谢谢我就行了，另外有空能多访问下我们的博客:blog.singcn.cn

不在废话了，看代码：

$(function() {
    //获取当前完整时间
    var now = new Date();
    //获取当前年份
    var vYear = now.getFullYear();
    //获取当年月份，因为js的月份是从0开始的，因此要加1
    var vMon = now.getMonth() + 1;
    //获取当前天
    var vDay = now.getDate();
    //声明执行时间，我此处定义为 当天的13:58:00开始执行
    var signOut = new Date(vYear + "/" + vMon + "/" + vDay + " 13:58:00");
    //相减获取毫秒数
    var time = signOut - now;
    //判断时间
    if (time > 0) {
        //延时执行DoSomeThings函数
        setTimeout("DoSomeThings()", time);
    }
});
//要定时执行的方法
function DoSomeThings() {
    alert("done!")
}
