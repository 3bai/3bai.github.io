落月VIP核心成员交流群拟成立啦。

发表于 2015-11-18   |   分类于 设计   |     |  

 

近日受群里大家点拨，突发奇想，准备成立一个核心vip群。
人数在10-100人之间。

性质呢，如同大学的社团一样。

下面是大概信息，征求下各位意见。

纲领

本群是一个交流性质的内部群，大家可以在此交流生活，游戏，学习，娱乐。为保证人人平等且不会随意踢人，群里不会设立管理员。

成员要求

  1. 初中（及以上）文化水平，素质高
  2. 落月粉丝群成员
  3. 有落月系列作品使用经验
  4. 热爱和谐，不得为喷子
  5. 遵守保密协议，禁止外传群内资源

成员福利

  1. 和大家交流学习的机会。
  2. 可以无偿使用群内其他人共享的优质内部资源。
  3. 落月现有（包含以后）作品永久免费使用。
  4. 落月部分作品仅对内部群发布。
  5. 部分作品源码\思路分享。

入群费

  1. 入群需要缴纳一定的费用，这些费用用于以下用途。
  2. 奖励群内发布优质资源的网友。
  3. 群内活动经费。
  4. 群内红包池。
  5. 本人作为群主不会动用群费。

群成立时效

  1. 11.18日筹备，11.25建群。明年11.25解散。
  2. 假如11.25前报名人数不足10人的，本计划取消。
  3. 假如明年11.25续费人数达到10人以上的，本群不会解散。
  4. 11.25日后不再收人/或收取双倍入群费

其他说明

  1. 落月切牌器不会作为仅对内群发布的作品，但是可能不会继续发布在本博客。即发布在粉丝群里。
  2. 内部群成员可以用到有特别定制且功能加强的切牌。
  3. 去后摇工具暂时停止更新，因为准备修改成连招编辑器。修改完成后会仅对内部群发布。
  4. 流浪自动连招会仅对内部群发布。
  5. 落月盒子会仅对内部群发布。
  6. 有意向可以回复报名~
  7. 顺便可以提下入群费多少的问题。

再次说明：落月切牌和以前一样继续免费更新，不会有任何变动。
